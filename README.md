
## Setup
Contains the .vim folder and .vimrc files that I use. 

Run the following commands in the terminal :  
    `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`


Then run the following in Vim :  
    `:PluginInstall`

## Plugins
